# chain-analyzer

A 股产业链全景分析工具。输入股票代码，自动生成带实时板块涨跌的产业链全景图。

## 功能

- **产业链全景**：上游原材料 → 中游制造（公司位置）→ 下游应用，三层结构一目了然
- **实时涨跌**：每个节点绑定同花顺板块指数，🔴红涨 🟢绿跌，一眼看清产业链冷暖
- **基本面选股**：按 PE/PB/股息率等条件筛选价值标的，按产业链归类展示
- **深色主题 HTML**：自包含页面，无需任何外部依赖

## 快速使用

```bash
# 安装依赖
pip install tushare

# 设置 Tushare Token
export TUSHARE_TOKEN="your_token_here"

# 生成产业链全景
cd scripts
python3 render.py 600580.SH

# 基本面选股
python3 value_screener.py
```
