## 🎯 功能概览

### 产业链全景图
输入任意 A 股股票代码，自动生成三层产业链结构：

```
⬆ 上游 · 原材料/零部件     ⬡ 中游 · 制造（公司位置）     ⬇ 下游 · 应用/消费
```

**每个节点附带对应板块的实时涨跌幅**，🔴 红涨 🟢 绿跌，一眼看清产业链冷暖。

### 深色主题 HTML 输出
- 自包含页面，零外部依赖
- 深色主题，卡片式布局
- 移动端自适应
- 可直接在浏览器打开或分享

## 🚀 快速开始

### 前置条件

- Python 3.8+
- [Tushare](https://tushare.pro/) 账号与 API Token

### 安装

```bash
# 克隆仓库
git clone https://github.com/benstartnow/chain-analyzer.git
cd chain-analyzer

# 安装依赖
pip install tushare

# 配置 Token
export TUSHARE_TOKEN="your_tushare_token_here"
```

### 运行

```bash
cd scripts
python3 render.py 600580.SH
```

输出文件：`{股票名}_产业链全景_{代码}.html`

## 🔌 数据源配置

本工具默认使用 **Tushare Pro**。以下列出可选的免费/低门槛数据源：

### 方案一：Tushare Pro（推荐）

| 项目 | 说明 |
|------|------|
| 官网 | [tushare.pro](https://tushare.pro/) |
| 费用 | 注册免费，基础接口免费 |
| 注册 | 手机号注册 → 个人主页 → 接口TOKEN |
| 本工具用法 | `export TUSHARE_TOKEN="你的token"` |

**用到的接口**（均为免费）：
- `stock_basic` — 股票信息
- `daily` — 日线行情
- `stock_company` — 公司介绍
- `ths_daily` / `ths_index` — 同花顺板块指数

### 方案二：AKShare（纯免费）

```bash
pip install akshare
```

将 `chain_data.py` 中的 Tushare 调用替换为对应的 AKShare 函数即可。

### 方案三：Baostock（纯免费）

```bash
pip install baostock
```

### 如何选择？

| 你的情况 | 推荐方案 |
|---------|---------|
| 想开箱即用 | **Tushare**（注册 1 分钟） |
| 不想注册 | **AKShare**（pip 即用） |
| 量化回测 | **Baostock** |

## 📁 文件结构

```
chain-analyzer/
├── README.md                    # 项目文档
├── SKILL.md                     # AI 技能定义
├── LICENSE                      # MIT 许可证
├── .gitignore                   # Git 忽略规则
└── scripts/
    ├── chain_data.py            # 产业链知识库（25+行业预设）
    └── render.py                # 产业链全景 HTML 渲染器
```

### 核心模块说明

| 文件 | 职责 |
|------|------|
| `chain_data.py` | 产业链知识图谱 + Tushare 数据获取 + 板块涨跌查询 |
| `render.py` | 深色主题 HTML 生成，每个节点带涨跌标签 |
