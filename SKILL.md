# chain-analyzer

## 描述
根据 A 股股票代码，自动生成该公司的**产业链全景图**——展示上游原材料/零部件、中游制造环节（公司所在位置）、下游应用领域的完整三层结构，**每个节点附带对应板块的实时涨跌幅**，一眼看清产业链冷暖。

## 触发条件
用户提到：
- "分析 [股票名/代码] 的产业链"
- "[股票] 上游有什么 / 下游应用"
- "产业链全景 / 产业链分析"
- "这个公司在产业链的什么位置"

## 执行流程

### Step 1：获取数据
运行 `scripts/render.py` 中的 `generate_report(ts_code, output_dir)`：
- 通过 Tushare 获取股票基本信息（名称、行业分类、最新股价）
- 根据行业分类匹配预设的产业链知识图谱
- 通过同花顺板块指数接口批量获取每个节点的板块涨跌幅

### Step 2：渲染输出
- 使用内置 HTML 模板生成深色主题产业链全景图
- 三层节点卡片：上游 → 中游（高亮公司位置）→ 下游
- 每个节点显示：名称 + 描述 + **板块涨跌幅标签（🔴红涨 🟢绿跌）**
- 顶部显示公司最新股价和涨跌幅

### Step 3：展示结果
- 打开生成的 HTML 文件
- 向用户简要说明产业链结构和涨跌分布
